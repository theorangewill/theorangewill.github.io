UNICAMP - Universidade Estadual de Campinas

Monitoring and optimizing the cost of HPC applications on the cloud

My master's research is focused on optimizing the cost of HPC applications in cloud computing based on the monitoring of computing resources and the containment of waste. With the guidance of Edson Borin and Marcio Roberto Miranda Assis, I proposed a methodology in which VMs are categorized in different states based on their behavior and use of resources; in this way, the system avoids waste with different reactions (such as a simple notification or stopping the VM). The system quantifies the waste of each VM based on the use of resources and in relation to an instance that optimizes cost and utilization. As we are working in an HPC environment, the application's performance should not be degraded. So, to select the instance that optimizes cost and usage and does not degrade performance, I proposed eight heuristics. I also developed a monitoring system architecture that allows to deal with the dynamics of the cloud.

With this work, I started working with cloud computing and learned how different services and providers work. I learned more about how to integrate APIs into my code and new Python libraries. I also improved my English and my writing, but mainly the research methodology; I learned more about systematic review and science in general.

At the moment, I have published an article about the monitoring system architecture and a report with the review of monitoring tools.

This research was part of a Petrobras project that aimed to optimize the use of HPC in cloud computing.

Otimização e Paralelização do Algoritmo de Roteamento de Lee

This was my undergraduate thesis and as advised by Nahri Balesdent Moreano.

I studied, optimized and parallelized the Lee's algorithm. This algorithm is a solution to the maze routing problem based on breadth-first search. I implemented the classic version, applied known optimizations proposed by Acker and Sait, and developed different parallelized solutions.

In this work, I learned more about C ++, OpenMP and parallel programming techniques, but I also improved my writing and my knowledge of the research methodology.

We published two articles with this work.

RESEARCHES

UFMS - Universidade Federal de Mato Grosso do Sul

Aug. 2017 - July 2018
Meta-heuristics for the Capacitated m-Ring-Star Problem

This was my Scientific Initiation during my graduation. Advised by Edna Ayako Hoshino.

In this research I developed meta-heuristics for the Capacitated m-Ring-Star Problem (CmRSP). I proposed two solutions: one using Greedy Randomized Adaptive Search Procedure (GRASP), the other using Variable Neighborhood Search (VNS).

This solution was integrated to the exact solution using Integer Linear Programming proposed by my advisor in her PHD.

With this work, I participated in my first conference, presenting a poster with the results at the National Congress of Applied and Computational Matheumatics.

I learned more about research methodology and started to work with meta-heuristics.

Apr. 2017 - current
ILP solution for the University Course Timetabling Problem

This was my first contact with research. With Edna Ayako Hoshino's guidance, I developed a system that solves the University Course Timetabling Problem using Integer Linear Programming (ILP). This system is used by the university's computing department to create the timetable of all courses.

With this work, I started to learn ILP, which became one of my interests. Today, we are still working to optimize the system with meta-heuristics and generalize it to other departments.
